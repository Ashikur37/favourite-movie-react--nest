<h3>
    Migrate and seed database
    </h3>
    <pre>
    yarn prisma db push
    yarn prisma db seed
    </pre>

<h3>
    Run backend server in dev mode
</h3>
<pre>
    Run dev database :
    docker compose up dev-db -d
    Run backend server in dev mode:
    yarn start:dev
</pre>

<h3>
    Run frontend server in dev mode
</h3>
<pre>
    yarn start
</pre>

<h3>Backend End to End Testing </h3>
<pre>

</pre>
